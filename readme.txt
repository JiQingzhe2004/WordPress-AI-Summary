使用爱奇吉摘要自动生成文章摘要和SEO优化内容，支持流式输出效果。

== 描述 ==

爱奇吉摘要是一款强大的WordPress插件，能够自动为您的文章生成智能摘要和SEO优化建议。通过先进的AI技术，帮助您提升内容质量和搜索引擎排名。

= 主要功能 =

* 自动生成文章摘要
* 支持流式打字机效果
* SEO优化建议
* 自定义摘要长度
* 响应式设计

= 技术特点 =

* 基于先进的AI模型
* 实时流式输出
* 高度可定制
* 性能优化
* 安全可靠

== 安装 ==

1. 在WordPress后台进入"插件 > 安装插件"
2. 搜索"爱奇吉摘要"
3. 点击"现在安装"
4. 安装完成后点击"启用插件"

== 常见问题 ==

= 插件支持哪些语言？ =

目前插件仅支持中文。

= 如何自定义摘要长度？ =

在插件设置页面可以调整摘要的长度参数。

= 是否支持自定义样式？ =

是的，您可以通过CSS自定义摘要的显示样式。
